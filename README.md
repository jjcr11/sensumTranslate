## Installation

Use the package manager [pip3](https://pip.pypa.io/en/stable/) to install the prerequisites.
### Prerequisites

```bash
python3
pip3

pip3 install google-cloud-translate==2.0.1
pip3 install pandas
pip3 install openpyxl
```
